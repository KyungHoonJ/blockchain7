이미지 태그
속성(attribute) : style, src, alt, name 같은 것들
속성은 태그를 설정
속성을 설정할 때는 속성명="" / 속성명=''
style 속성의 경우 CSS 문법에 맞춰 작성한다.
style 속성 = CSS
width : 너비, max-width, min-width
height : 높이, max-height, min-height
background-color : 배경색, 색상 자체를 입력할 수 있고, rgb(***,***,***) 형식으로 가능(***는 0~255)
padding(내부 여백), margin(외부 여백) : ***-top, ***-right, ***-bottom, ***-left 형식으로 하나씩 설정 가능
box-sizing : 박스의 크기를 어떻게 설정할지 기준을 설정
display : 영역의 노출 크기를 어떻게 설정할지 기준을 설정한다.
block은 부모를 기준으로 모든 너비를 차지한다, 높이는 자식을 기준으로 한다.
부모란 상위 태그이다.
